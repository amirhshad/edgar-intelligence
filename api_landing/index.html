Quick Start

Two lines to your first query

cURL 
$ curl -X POST https://api.edgarintel.com/v1/query \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "What are Microsoft biggest risk factors?",
    "company": "MSFT",
    "filing_type": "10-K"
  }'
Python 
import requests

response = requests.post(
    "https://api.edgarintel.com/v1/query",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "question": "What are Microsoft biggest risk factors?",
        "company": "MSFT",
    }
)

data = response.json()
print(data["answer"])

for cite in data["citations"]:
    print(f"  [{cite['index']}] {cite['section']}")
